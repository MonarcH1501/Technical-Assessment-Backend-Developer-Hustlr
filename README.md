# E-commerce Product API

## Tech Stack
- Node.js
- Express.js

## How to Run
```bash
npm install
npm start
```

Server runs at: http://localhost:3000

## API Endpoints
- GET /products
- GET /products/:id
- GET /products?category=Apparel
- POST /products

See `/docs` folder for:
- Postman collection
- Swagger (OpenAPI) YAML
